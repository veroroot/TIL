# 기본 명령어

CLI(Command Line Interface) 명령어를 배워봅시다.

- `pwd` (Present Working Directory)
  - 현재 위치의 디렉토리(최상위 경로부터 현재경로까지 표시함) 
- `ctrl`+`L` 
  - 현재 커맨드 창 초기화
- `cd` (Change Directory)
  - 디렉토리(폴더) 위치 변경
  - cf) /는 최상단(root)을 의미함
  - cf) ~는 사용자(home)를 의미함
  - `Tap`+`Tap`키를 연속으로 누르면 자동완성
- `ls`(List)
  - 현재 위치의 폴더 및 파일을 출력
